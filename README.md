# PRAKTIK-DETEKSI-TEPI-ROBERT-SOBEL

Repositori ini berisi implementasi program python untuk melakukan deteksi tepi operator roberts cross dan sobel

library yang digunakan :
- numpy
- imageio
- matplotlib

perbandingan hasil antara roberts dan sobel :
1. roberts memiliki kernel yang sangat kecil jadi lebih sensitif terhadap noise sedangkan sobel memberikan hasil lebih halus dan stabil
2. dari sisi ketajaman, roberts menghasilkan tepi yang lebih tajam  sedangkan sobel memberikan tepi lebih tebal dan halus
3. kinerja sobel lebih cepat sedangkan lebih lambah tapi hasilnya lebih akurat
